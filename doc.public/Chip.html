Module Chip

module Chip: sig .. end

Describes the chips in the casino and operations that you can do on them

type t

abstract type representing a chip

type color =
| White of int
| Red of int
| Blue of int
| Green of int
| Black of int

options for color type, white is always worth 1, Red worth 5, Blue worth 10, Green worth 25, and Black worth 100, the int value is the quanity of chip

exception RI_Broken
exception Not_Within
val repo_ok : t -> t

RI: type t must in the form White, Red, Blue, Green, Black. Returns type t if RI is okay and a failure mesage otherwise.

val empty : t

empty chip representation

val get_value : t -> int

get_value t is the integer representation of the value of the chips

val simplify : t -> int -> t

simplify t steps simplifies the chips by the single_step_simplify function steps times or until it can no longer be simplified.

val break : t -> int -> t

break t steps breaks larger denominations using he single_step_break function steps times or until it can no longer be broken.

val add : t -> t -> t

add t1 t2 is the combined quantities in t2 to t1. Raises: RI_Broken if the Representation invariant is broken

val is_within : t -> t -> bool

is_within c1 c2 is true if c2 is a subset of c1 by specific color chip

val bet : t -> t -> t

bet t1 t2 removed t2 from t1 Returns: t1 with t2's quantities removed Requires: t2 be a subset of t1 Raises: Not_Within if t2 is not contained within t1, or RI_Broken if the representation invariant is broken

val create_chips : int -> int -> int -> int -> int -> t

create_chips w r b g bl returns the chip type with the specified number of chips for each color.

val to_string : t -> string

to_string t returns a string of the inputted chips

val create_bot_bet : int -> t -> t

create_bot_bet i p creates a bet valid for min_bet i and player p

val create_bot_chips : int -> t -> t

create_bot_chips i p creates a chip for bot that is valid with min_bet i
